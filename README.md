# 📝 TextUtils – React Text Manipulation App

**TextUtils** is a simple, responsive React-based text utility app that enables users to manipulate text in various ways. It's ideal for developers, writers, and students who want to clean, format, or analyze their text instantly.

🔗 **Live Demo**: [TextUtils React App](https://tejas-santosh-nalawade.github.io/TextUtils-react/)

---

## 🚀 Features

- ✅ Convert to **Uppercase**
- ✅ Convert to **Lowercase**
- ✅ **Remove Extra Spaces**
- ✅ **Copy Text to Clipboard**
- ✅ **Clear Text**
- ✅ **Word & Character Count**
- ✅ **Estimated Reading Time**
- 🌙 Toggle between **Light & Dark Modes**
